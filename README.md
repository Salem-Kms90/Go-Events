# GoEvents - Cinematic Experience

Premium multi-page website for GoEvents, featuring a cinematic design and immersive interactions.

## Technical Stack

- **Engine:** Vite
- **Styling:** Tailwind CSS v4 (Modern & Hybrid approach)
- **Logic:** Vanilla JavaScript (ESNext)
- **Design System:** Midnight Abyss (#05070A) & Gold Champagne (#D4AF37)
- **Typography:** Onest (Sans-serif)

## Key Features

- **GoCiné:** Immersive streaming-style gallery with video previews and detail modals.
- **Ticketing Widget:** Floating multi-step purchase flow.
- **Responsive Design:** Mobile-first architecture with smooth transitions.
- **Performance 2026:** Optimized assets and minimal runtime overhead.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

- `index.html`: Home page
- `gocine.html`: Immersive gallery
- `services.html`: Service offerings
- `portfolio.html`: Showcase of works
- `contact.html`: Contact and booking
- `js/main.js`: Main logic and components
- `css/style.css`: Custom styles and Tailwind directives
